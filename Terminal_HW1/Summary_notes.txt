#This file contains homework excercises and it's solutions.
1) Посмотреть где я - pwd
2) Создать папку - mkdir 'foldername'
3) Зайти в папку - cd 'foldername'
4) Создать 3 папки - mkdir fn1 fn2 fn3
5) Зайти в любоую папку - cd 'foldername'
6) Создать 5 файлов (3 txt, 2 json) - touch 'fln1.txt'x3 'fln1.json'x2 
7) Создать 3 папки -  mkdir 'fn1_1' 'fn2_1' 'fn3_1'
8. Вывести список содержимого папки - ls -l
9) + Открыть любой txt файл - vim 'fln1.txt'
10) + написать туда что-нибудь, любой текст. - i
11) + сохранить и выйти. - :wq
12) Выйти из папки на уровень выше - cd ..
13) переместить любые 2 файла, которые вы создали, в любую другую папку. - mv fn1 fn2 ~/path
14) скопировать любые 2 файла, которые вы создали, в любую другую папку. - cp fn1 fn2 ~/path
15) Найти файл по имени - find -name 
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает. - grep
17) вывести несколько первых строк из текстового файла - head -n
18) вывести несколько последних строк из текстового файла - tail -n
19) просмотреть содержимое длинного файла (команда less) изучите как она работает. - less -s
20) вывести дату и время - date
